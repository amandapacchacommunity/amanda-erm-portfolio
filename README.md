# Amanda ERM Portfolio

## Overview
This project is a demonstration of an Enterprise Risk Management (ERM) portfolio system.  
It showcases how risk registers, case studies, and risk analytics can be organized and presented in a simple web-based interface.

The project highlights skills in:
- Risk management concepts
- Data organization
- Web development (HTML, CSS, JavaScript)
- Basic risk analytics

---

## Features
- Risk register stored as structured data
- Case studies for risk scenarios
- Interactive web interface
- Organized project structure for ERM workflows

---

## Project Structure

amanda-erm-portfolio

├── case-studies/ – Risk scenario documentation

## Example Risk Data

The project includes a sample risk register dataset containing structured risk information such as:

- Risk ID  
- Risk description  
- Impact level  
- Likelihood  
- Mitigation strategy  

This demonstrates how ERM systems organize risk information.
